Installer l’application W.S Corporation

Guide d’installation – Sélectionnez votre appareil pour suivre les étapes adaptées.

Android
  • Ouvrez Google Chrome
  • Visitez wscorporation.netlify.app
  • Une bannière “Ajouter à l’écran d’accueil” apparaît
  • Appuyez sur Ajouter
  • Cliquez sur “Installer”
iPhone / iPad
  • Ouvrez le site dans Safari
  • Appuyez sur Partager ▢↑
  • Choisissez “Ajouter à l’écran d’accueil”
  • Appuyez sur Ajouter
PC (Windows)
  • Ouvrez le site dans Chrome ou Edge
  • Regardez la barre d’adresse
  • Cliquez sur “Installer” (icône flèche)
  • L’application s’installe sur votre bureau
MacBook
  • Sur Safari : menu Fichier → Ajouter au Dock
  • Sur Chrome : icône Installer dans la barre d’adresse
  • L’app apparaît dans le Dock / Applications
